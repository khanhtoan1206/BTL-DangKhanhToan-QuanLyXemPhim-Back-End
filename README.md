# BTL - Quản Lý Xem Phim (Back-End)

## Giới thiệu
Đây là project **Back-End của hệ thống quản lý rạp chiếu phim** được xây dựng để phục vụ cho bài tập lớn.  
Hệ thống cho phép quản lý các thông tin như phim, phòng chiếu, ghế, suất chiếu và người dùng.
